About Me

Hello! I'm Ronak Gujarathi, a dedicated and curious Electronics and Telecommunication (ENTC) graduate from Vishwakarma Institute of Information Technology, Pune. Throughout my academic journey, I've developed a strong interest in software development, particularly in building modern web applications and intuitive user experiences.

My technical toolkit includes languages and tools such as Python, Java, C/C++, HTML, CSS, JavaScript, Bootstrap, and MySQL. I enjoy combining these skills to develop responsive and user-friendly websites with both frontend and backend logic.

I began exploring full-stack development during my first semester, when I interned at Gudel India Pvt. Ltd.. This hands-on experience helped me understand the industry’s best practices and the importance of clean code, teamwork, and continuous learning.

I’m currently focused on sharpening my skills in full-stack development, contributing to meaningful projects, and preparing myself for real-world opportunities in the tech industry. I believe in building software that not only functions efficiently but also creates a positive impact for its users.

Projects

Skin Disease Prediction using VGG16

A deep learning-based project that uses VGG16 to classify skin diseases...
This model is trained on a dataset containing thousands of skin disease images. The VGG16 architecture is fine-tuned for multi-class classification of skin conditions such as acne, eczema, psoriasis, etc. The model achieves high accuracy using transfer learning. The frontend allows users to upload an image, and the backend returns a diagnosis. The system is designed for medical assistance and educational use.

Dynamic HMI Navigation System

A browser-based simulation of an HMI (Human-Machine Interface) screen that...
This project mimics the TIA Portal HMI Text List screen. It displays live data fetched from a Siemens PLC via WebSocket/JS integration. Each value corresponds to a different screen view or state. Buttons dynamically change states, enabling efficient testing of industrial interfaces without needing a physical HMI. The goal is to simplify the debugging process and create interactive UIs for automation systems.

Forever – Buy, Sell, Recycle Platform

A sustainable fashion marketplace built using full-stack technologies...
Forever is a web platform that promotes circular fashion. Users can buy second-hand clothes, sell their unused items, or recycle them. The backend (Node.js + MongoDB) manages user data, payments, and product listings. The frontend uses React for a responsive experience. It supports authentication, cart system, product filters, and admin panel. This platform addresses fashion waste through tech.
